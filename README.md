# Ferretto Capital Website

Static website for Ferretto Capital — a professionally-run Family Office focused on generating consistent long-term returns by allocating into a globally diversified portfolio of funds, with a focus on alternatives.

## Stack

Plain HTML, CSS, and a little JavaScript. No build step. Fonts (Montserrat) are self-hosted in `assets/fonts/`.

## Local preview

Open `index.html` in a browser — that's it.

## Deployment

Served via GitHub Pages from the `main` branch of this repo.
